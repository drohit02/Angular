String pipes

{{str}} : Without using Pipe

{{str | uppercase}} :String after using uppercase pipe

{{str | lowercase}} :String after using lowercase pipe

{{str | slice:0:5}} :String after using [slice:from index : to given index (slice:0:5)] param pipe

Number Pipe

1. Decimal Pipe

{{pi}} = Number Without pipe

{{pi | number:"0.1-2"}} = number after number pipe

2. Currency Pipe

{{newCurrency | currency:"USD"}} : Dollar

{{newCurrency | currency:"EUR"}} :Euro

3. Percentile Pipe

{{marks}} without currency pipe

{{marks | percent : "2.2-2"}} with currency pipe
